# 🖤 Sandrone-MD

> *"Soy una presencia silenciosa... estoy aquí para acompañarte con calma y discreción."*

Bot de WhatsApp elegante y misterioso, creado desde cero en Termux sin necesidad de PC.

---

## ✨ Características

- 🕯️ Personalidad elegante y amable
- 🌸 Bienvenida automática con foto de perfil
- 🛡️ Sistema antispam con 3 advertencias
- 🎭 Comandos para administradores
- 🔒 Comandos exclusivos del propietario
- 📸 Convertidor de imágenes a stickers

---

## 📋 Comandos

### Generales
| Comando | Descripción |
|---------|-------------|
| #menu | Menú principal |
| #infomenu | Guía detallada de comandos |
| #info | Información sobre Sandrone |
| #hora | Hora actual |
| #dado | Número del destino (1-6) |
| #saludo | Saludo especial |
| #s | Convierte imagen a sticker |

### Administradores
| Comando | Descripción |
|---------|-------------|
| #ban @usuario | Expulsa a un miembro |
| #agregar número | Agrega a alguien al grupo |
| #anuncio mensaje | Anuncio oficial al grupo |
| #onantispam | Activa el antispam |
| #offantispam | Desactiva el antispam |

### Propietario
| Comando | Descripción |
|---------|-------------|
| #apagar | Apaga el bot |

## ⚙️ Instalación en Termux

```bash
# 1. Instalar dependencias
pkg install nodejs git

# 2. Clonar el repositorio
git clone https://github.com/Ofc-Nasuma/Sandrone-MD
cd Sandrone-MD

# 3. Instalar paquetes
npm install
npm install --cpu=wasm32 sharp

# 4. Iniciar
node index.js
```
